HOW TO USE THESE FILES
================================================================================

OPTION 1: Run Locally
  1. Create folder: Brain-MRI-Segmentation/
  2. Create subfolders: models/, scripts/, utils/, app/
  3. Copy files into appropriate folders:
     - models_*.py → models/
     - scripts_*.py → scripts/
     - utils_*.py → utils/
     - app_*.py → app/
  4. Copy: requirements.txt, README.md, .gitignore, LICENSE
  5. Create data/ folder for dataset
  6. Run: python scripts/train_local.py

OPTION 2: Use with Kaggle
  1. Upload files to Kaggle dataset
  2. Create Kaggle notebook
  3. Import training script
  4. Run: python train_kaggle.py

OPTION 3: Deploy with Streamlit
  1. Install dependencies: pip install -r requirements.txt
  2. Run: streamlit run app/frontend.py
  3. Access: http://localhost:8501
